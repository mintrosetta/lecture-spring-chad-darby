
    :: Spring Overview

    :: Core Container
    > Beans
        - class ที่ถูกจัดการด้วย spring
    > Core
        - จัดการการสร้าง bean 
    > SpEL
        - spring expression language
        - เป็นภาษาเล็ก ๆ ที่เราใช้ในการ config file
    > Context
        - เป็น spring container จัดเก็บ bean ไว้ใน memory
